// OVERVIEW SECTION STARTS
[#overview]

[float]
=== Description
Stops the keyboard emulation to a connected computer. To start keyboard emulation, use link:../keyboardBegin[Keyboard.begin()].
[%hardbreaks]


[float]
=== Syntax
`Keyboard.end()`


[float]
=== Parameters
Nothing

[float]
=== Returns
Nothing
